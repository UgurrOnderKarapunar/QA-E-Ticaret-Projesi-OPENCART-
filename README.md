# QA-E-Ticaret-Projesi-OPENCART-
Bu proje, manuel QA testi kapsamında kullanıcı kayıt, giriş, ürün arama, sepet, ödeme ve sipariş takibi fonksiyonlarının test edildiği bir örnek projedir. 

**İçerik:**
- İş Gereksinimleri
- Kullanıcı Hikayeleri ve Kabul Kriterleri
- Test Senaryoları (Pozitif & Negatif)
- Bulunan Hatalar ve Önceliklendirme
